# MaidThis — UI Kit: Franchisee Ops Portal

Operator-facing dashboard for franchise owners. Covers:
- Dashboard KPIs, today's visits
- Job schedule board
- Cleaner roster + performance
- Booking detail drawer

## Files
- `index.html` — clickable shell with sidebar nav + 3 views
- `Sidebar.jsx`, `TopBar.jsx`
- `DashboardView.jsx`, `ScheduleView.jsx`, `TeamView.jsx`
- `JobCard.jsx`, `KpiCard.jsx`, `BookingDrawer.jsx`
